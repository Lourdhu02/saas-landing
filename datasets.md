# Datasets & Assets

## Icon Sets
- Lucide (most common in modern Next.js/Tailwind projects)
- Phosphor Icons (consistent weight, popular in SaaS)
- Radix Icons (primitive, design-system-native)

## Design Tokens (adopt CSS custom properties approach)
- Shadow system: --shadow-hue → primitives → --clay-sm/md/lg tokens
- Palette: off-white base, near-black dark, single saturated accent + pastel set
- Typography: Inter (UI), JetBrains Mono (display/code), PP Neue Montreal (premium)

## Fonts
- Google Fonts: Inter (self-host for Lighthouse, no external calls)
- premium: Aeonik, GT Super, Söhne, ABC Diatype (via @font-face or CDN)
